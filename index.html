This is a simple map showing a selection of yakiniku (grilled meat) restaurants in Fukuoka, Japan. Each map marker is coded by an alphabetical letter, which then can be found in the selection display area below the map. Each restaurant display card also has a Google Plus code included, which can be entered into a Google Map search and take you to the exact location.

About the Yakiniku Listings

This is a selection of yakiniku restaurants in Fukuoka, Japan. Not every location is listed, but just a selection to make creating the demonstration map and display grid easier. A regular Google Maps search will show a much larger listing of great yakiniku restaurants!
